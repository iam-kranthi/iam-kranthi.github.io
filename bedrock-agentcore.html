Architecture · 03

A layered runtime from request boundary to telemetry.

The deployment separates authorization, runtime execution, workflow orchestration, business tool access, and observability. Each concern can evolve without changing the diagnosis contract.

Operational design · 04

Production concerns become first-class architecture.

The runtime adds security, deployment consistency, monitoring, and model resilience around the same diagnosis workflow used locally.

Runtime boundary

Containerized execution inside Bedrock AgentCore

Docker packages the LangGraph application and its dependencies into a repeatable unit. AgentCore provides the managed execution boundary for invocation, scaling, and managed AI operations.

Security

IAM-authorized requests

Runtime access is controlled through AWS identity and authorization rather than an open application endpoint.

Observability

CloudWatch telemetry

Logs, traces, and runtime signals make workflow behavior visible during production diagnosis.

Integration

MCP tools stay reusable

The product-access layer remains separate from deployment and can be reused by future agents.

Roadmap

Foundation for multi-agent operations

  • Add a Supervisor Agent for routing and policy control
  • Run Catalog, Inventory, Pricing, and Summary agents in the same runtime model
  • Expand CloudWatch telemetry to agent-level traces and tool-call metrics
  • Preserve local generation with model fallback for response resilience
Architecture highlights

Operational concerns represented in the codebase

  • Containerized LangGraph runtime with a repeatable deployment boundary
  • IAM-authorized invocation instead of an open application endpoint
  • MCP tools isolated from runtime and orchestration concerns
  • CloudWatch logging and telemetry around workflow execution
Engineering learnings

What deployment design reinforced

  • Security and observability must be architecture inputs
  • Packaging the workflow clarifies runtime dependencies
  • Deployment should not change the diagnosis contract

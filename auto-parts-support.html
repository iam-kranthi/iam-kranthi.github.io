Auto Parts Technical Knowledge Platform

An Enterprise RAG platform for auto-parts technical knowledge. The application ingests local HTML articles, extracts and chunks technical content, creates HuggingFace embeddings, stores vectors in ChromaDB and delivers grounded AI answers using a Local LLM with Cloud LLM fallback.


FastAPI RAG ChromaDB HuggingFace Embeddings Local LLM Cloud LLM Fallback LangChain

Business Problem

Auto-parts technical knowledge is often spread across HTML articles, manuals and support documents. Users need quick, accurate answers without manually searching many pages.

AI Solution

The system extracts technical content from HTML files, chunks the text, creates HuggingFace embeddings, stores vectors in ChromaDB and retrieves relevant context for AI-powered question answering with local and cloud LLM fallback.

Business Value

Improves technical support speed, makes knowledge searchable, reduces manual lookup effort and creates a reusable RAG foundation for auto-parts assistance.
